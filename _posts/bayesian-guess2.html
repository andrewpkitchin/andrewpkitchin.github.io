Here's a simplified version of how it works:

Suppose your prior belief is that all numbers are equally likely (1% each). Now suppose you get a clue, say c = 10. If a number i is the true number, then the chance of getting this clue is P(c | n = i), which is 1/i if c ‚â§ i, and 0 otherwise.

Using Bayes' theorem: $$ P(n = i \mid c) \propto P(c \mid n = i) \cdot P(n = i) $$ We multiply the prior by the likelihood and normalize across all values to update our beliefs.
